# Doc

## 作用：
* 能够实时显示选择的客户端所建立的每一条网络连接，并且实时显示每条连接访问的数据包的总大小。
* 能够解析并查看tcp和http的包内容
* 默认对抓包数据进行保存，并可查看保存的记录

首先选择要抓包的应用，也可以选择抓手机所有的包：
