# Doc

## 作用：
* 能够实时显示选择的客户端所建立的每一条网络连接，并且实时显示每条连接访问的数据包的总大小和总个数。
## 步骤
* 选择所有抓取数据包的客户端。
* 打开VPN。
## Function:
*It shows the state of Network Connection which is established by your selected client.
